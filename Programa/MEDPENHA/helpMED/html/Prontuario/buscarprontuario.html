COMO BUSCAR O PRONTUÁRIO


Para se buscar o prontuário é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Agenda"
  2. Após aparecerá um sub-menu, selecione o item " Visualizar Agenda".
  3. Ao selecionar o sistema mostrara uma tela exibindo os paciente marcado para atendimento daquele dia
  4. Selecione a linha referente ao paciente que deseja busar seu prontuário
  5. Clique no botão " Buscar Prontuário".
