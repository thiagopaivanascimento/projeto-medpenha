COMO VISUALISAR A AGENDA MÉDICA


Para se visualisar a agenda médica é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Agenda"
  2. Após aparecerá um sub-menu, selecione o item " Visualizar Agenda".
  3. Ao selecionar o sistema mostrara uma tela exibindo os paciente marcado para atendimento daquele dia
  4. Em seguida você terá a lista de todos as horário daquele dia.
