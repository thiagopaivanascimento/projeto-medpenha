COMO LISTAR CONSULTAS(CONFIRMAR E CANCELAR)


Para se Confirmar e Cancelar consultas é necessário ser o usuário responsável por essa função no sistema e estar devidamente logado no mesmo,e logo após seguir os seguintes passos:
  1. Clique no menu "Consulta"
  2. Após aparecerá um sub-menu, selecione o item "Listar Consultas".
  3. Ao selecionar o sistema mostrara uma tela com todas as consultas marcadas no sistema
  4. Selecione a linha referente a consulta que deseja confirmar ou cancelar
  5. Clique em "Confirmar" para que a consulta seja confirmada,ou Clique em "Cancelar" para que a consulta seja cancelada
