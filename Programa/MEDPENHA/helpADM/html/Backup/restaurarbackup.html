COMO SE RESTAURAR O BACKUP DO SISTEMA


Para se restaurar o backup do sistema é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Backup"
  2. Após aparecerá um sub-menu, selecione o item "Restaurar Backup".
  3. Em seguida você já irá ter restaurado o backup do sistema.
