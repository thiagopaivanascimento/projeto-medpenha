COMO EFETUAR O BACKUP DO SISTEMA


Para se efetuar o backup do sistema é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Backup"
  2. Após aparecerá um sub-menu, selecione o item "Realizar Backup".
  3. Em seguida você já irá ter feito o backup do sistema.
