COMO EMITIR O RELATÓRIO ESTATÍSTICO POR PAGAMENTO PARTICULAR


Para se emitir relatórios do sistema é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Relatórios"
  2. Após aparecerá um sub-menu, selecione o item "Relatório estatístico por pagamento particular".
  3. Ao selecionar o sistema mostrara uma tela pedindo a data referente ao relatório
  4. Em seguida você já irá poder emitir o relatório.
