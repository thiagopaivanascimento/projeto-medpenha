COMO SE VERIFICAR PACIENTES POR CONVÊNIO


Para se listar os pacientes pelo seu convênio no sistema é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Lista de Pacientes"
  2. Após aparecerá um sub-menu, selecione o item "Lista de Pacientes por Convênio".
  3. Ao selecionar o sistema mostrara uma tela indicando os convênios cadastrados no sistema
  4. Selecione o convênio e clique no botão "Listar Pacientes"
  5. O sistema exibirá todos os pacientes que possuem este convênio
  6. Em seguida você poderá emitir esta lista.
