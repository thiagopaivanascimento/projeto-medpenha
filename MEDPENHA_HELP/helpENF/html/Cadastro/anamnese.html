COMO EFETUAR O CADASTRO DE UMA ANAMNESE


Para se cadastrar uma anamnese é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Cadastro"
  2. Após aparecerá um sub-menu, selecione o item "Anamnese".
  3. Ao selecionar o sistema mostrara uma tela pedindo os dados referentes ao cadastro
  4. Preencha o campo matricula e clique em "Buscar Paciente",se o paciente estiver cadastrado, o sistema retornará as dados do paciente
  5. Preencha os demais campos e clique em "Salvar",o sitema também oferece as opções "Alterar","Consultar", "Limpar"e "Voltar"
  6. Em seguida você já irá ter efetuado o cadastro.

OBSERVAÇÃO: caso os campos obrigatórios não sejam preenchidos, o sistema mostrará uma mensagem e não realizará o cadastro
