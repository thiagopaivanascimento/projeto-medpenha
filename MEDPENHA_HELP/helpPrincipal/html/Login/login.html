COMO SE EFETUAR LOGIN NO SISTEMA


Para se logar no sistema é necessário que se digite o nome de usuário e senha corretos para que o sistema possa reconhecê-lo.
  1. Clique no menu "Login"
  2. Após aparecerá um sub-menu, selecione o item "Efetuar Login".
  3. Ao selecionar o sistema mostrará uma tela pedindo o nome de usuario e a senha cadastrados
  4. Sua permissão será de acordo com a sua função na clínica.
  5. Em seguida você já irá ter feito o login.

OBSERVAÇÃO: caso os campos não sejam preenchido o sistema mostrará uma mensagem e não realizará o login
