COMO EFETUAR O CADASTRO DE UM MÉDICO


Para se cadastrar um médico é necessário ser o usuário responsável por essa função no sistema, estar devidamente logado no mesmo, e logo após seguir os seguintes passos:
  1. Clique no menu "Cadastros"
  2. Após aparecerá um sub-menu, selecione o item "Médico".
  3. Ao selecionar o sistema mostrara uma tela pedindo os dados referentes ao cadastro
  4. Preencha os campos e clique em "Salvar",o sitema também oferece as opções "Alterar","Consultar","Deletar" e "Voltar"
  5. Em seguida você já irá ter feito o cadastro.

OBSERVAÇÃO: caso os campos obrigatórios não sejam preenchido o sistema mostrará uma mensagem e não realizará o cadastro
